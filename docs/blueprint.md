# **App Name**: Rupiyoo Connect

## Core Features:

- Borrower Loan Request Posting: Allows borrowers to post loan requirements with amount, location, and details.
- Lender Loan Request Discovery: Enables lenders to view nearby loan requests and connect with borrowers.
- User Verification Tool: AI-powered tool to perform KYC verification of lenders and borrowers.
- Real-time Notification System: Provides real-time updates and notifications to borrowers and lenders.
- Secure Messaging: A secure messaging platform for borrowers and lenders to communicate directly within the platform.
- Legal Document Library: AI powered tool that maintains a library of auto-updated legal documents related to digital lending guidelines for Rupiyoo's operating region
- Profile and Request Moderation: Tools to facilitate reporting/moderating users, along with reporting requests that are deemed as unwanted or unsafe to fulfill. This creates a safer, higher trust marketplace.

## Style Guidelines:

- Primary color: Teal (#4db6ac), offering a sense of trust and stability.
- Background color: Light gray (#f5f5f5), for a clean and modern look.
- Accent color: Blue (#29b6f6) , to complement the teal and enhance usability.
- Body and headline font: 'Inter', a sans-serif font for clarity and modernity.
- Note: currently only Google Fonts are supported.
- Use clean, line-based icons to represent features and actions.
- Mobile-first, fully responsive layout.
- Subtle animations on button hovers and section transitions to enhance user experience.
